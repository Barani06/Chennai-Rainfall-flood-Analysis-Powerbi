# Chennai Rainfall and Flood Level Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard that analyzes rainfall patterns and flood levels across different areas of Chennai. The dashboard helps identify rainfall trends, flood-prone regions, and variations in rainfall levels over time using data visualization techniques.

The analysis focuses on understanding how rainfall varies across months and years and how it impacts flood levels in different parts of the city.

## Key Features

* KPI cards displaying the **highest rainfall recorded** and **average flood level**
* **Rainfall trend analysis** over time using line charts
* **Flood level comparison by area** using bar charts
* **Monthly rainfall comparison** between minimum and maximum rainfall
* **Flood level distribution** using a donut chart
* **Geographical visualization** of flood levels using a map
* **Interactive filters** that allow users to analyze specific areas

## Dataset Information

The dataset includes rainfall and flood-related data for various areas of Chennai.

Columns used in the analysis:

* Date
* City
* Area
* Minimum Rainfall
* Maximum Rainfall
* Flood Level

## How to Use the Dashboard

1. Open the `.pbix` file using **Power BI Desktop**.
2. Use the **Area filter** to analyze specific locations.
3. Explore rainfall trends and flood levels through the visualizations.
